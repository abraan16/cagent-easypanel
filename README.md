# cagent para EasyPanel

Imagen Docker personalizada de cagent para EasyPanel.

## Configurar API Keys

Edita el archivo `.env` con tus claves reales.

## Usar en EasyPanel

2. Conecta el repositorio en EasyPanel
3. Configura las variables de entorno
